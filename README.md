## Prerequisites

- Node.js 12.x
- Yarn 1.22.4

## Available scripts

In the project directory, you can run:

### `yarn`

Installs dependencies.

### `yarn start`

Runs the application locally.

### `yarn storybook`

Runs the storybook locally.
